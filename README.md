### 📝 Project Overview
This repository contains a web blog entry for **"THE DIGITAL VILLAGE: WEB DESIGN COMPETITION"** held at **Bapu Primary-High School**. 

The competition challenged students from Class 7 and 8 to apply their knowledge of **HTML and CSS** to create a digital narrative about rural roots and heritage. Anmol's project, "Native Places of India," highlights the cultural heartbeat of various Indian regions, blending modern code with traditional storytelling.

### 💻 Technical Skills Demonstrated
* **HTML5 Semantic Structure:** Organized use of headers, containers, and image elements.
* **CSS3 Styling:** * Custom layout design using a 85% width centered container.
    * Color coordination (Teal backgrounds with Royal Blue headers).
    * Responsive image handling and typography for better readability.
* **Content Curation:** Researching and presenting information on diverse locations like Varanasi, Jaipur, Kerala, and Shillong.

### 🖼️ Preview & Setup
To view this project:
1.  Clone this repository or download `anmolentry.html`.
2.  Place the required images (e.g., `varansi.jpg`, `jaipur.jpg`, `manali.jpg`) in the same directory.
3.  Open the file in any web browser.
